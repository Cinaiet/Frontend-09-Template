### flex排版

### 主轴
表示页面排版时的排列的方向

```
flex-direction: row //  Main主轴: contains  x, width , left , right
                    //  Cross: y, height, top, bottom

flex-direction: column //  Cross 交叉轴: contains  x, width , left , right
                       //  Main: y, height, top, bottom



```
